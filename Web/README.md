# Web 前端项目（Next.js + Ant Design）

本目录包含智能轴承监测系统的前端部分，提供用户可视化界面、历史趋势图、状态告警等。

##  技术栈

- React + Next.js
- Ant Design（UI组件库）
- Axios（接口请求）
- Tailwind CSS（可选）
- echarts / AntV Charts（数据可视化）

##  项目结构

- `pages/`：路由页面定义
- `components/`：通用组件封装
- `services/`：请求后端 API 的接口封装
- `public/`：静态资源
- `styles/`：全局样式

##  本地开发启动

```bash
npm install
npm run dev
